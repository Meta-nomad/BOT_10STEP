# 🔵 Бот «10-й Шаг АН» для Telegram

Телеграм-бот для ведения ежедневной инвентаризации по 10-му Шагу программы Анонимных Наркоманов.

---

## Форматы инвентаризации

| Формат | Описание |
|---|---|
| 📖 Руководство по Шагам АН | Официальный формат с вопросами по чувствам, поступкам, отношениям |
| 📕 Базовый Текст (БТ) | Акцент на принципы выздоровления и честность |
| 📗 It Works (IW / Айпи) | Глубокая работа с принципами 10-го Шага |
| ☀️ Ежедневная (краткая) | 5 ключевых вопросов, 5–10 минут |
| 🌙 Вечерняя (расширенная) | Полный обзор дня: гнев, страхи, обиды, благодарность |
| 👤 Формат спонсора | Свободная запись — то, что расскажешь спонсору |

---

## Установка и запуск

### 1. Получи токен бота

1. Открой Telegram, найди **@BotFather**
2. Отправь `/newbot`
3. Введи имя и username бота
4. Скопируй выданный токен (вида `123456789:AAF...`)

### 2. Установи зависимости

```bash
pip install -r requirements.txt
```

### 3. Задай токен

**Способ А — через переменную окружения (рекомендуется):**
```bash
export BOT_TOKEN="твой_токен_здесь"
python bot.py
```

**Способ Б — прямо в коде:**  
Открой `bot.py`, найди строку:
```python
token = os.environ.get("BOT_TOKEN", "YOUR_BOT_TOKEN_HERE")
```
Замени `YOUR_BOT_TOKEN_HERE` на свой токен.

### 4. Запусти бота

```bash
python bot.py
```

---

## Использование

1. Найди своего бота в Telegram и отправь `/start`
2. Выбери *«Новая инвентаризация»*
3. Выбери формат
4. Отвечай на вопросы текстом
5. Нажимай *«Следующий вопрос»* или *«Пропустить»*
6. По завершении нажми *«Завершить»* — запись сохранится

---

## Данные

Все записи хранятся в файле `inventory_data.json` в той же папке, что и бот.  
Данные хранятся локально и никуда не передаются.

---

## Команды

| Команда | Действие |
|---|---|
| `/start` | Запустить / перейти в главное меню |
| `/cancel` | Отменить текущую инвентаризацию |

---

## Запуск на сервере (24/7)

Для постоянной работы можно использовать `systemd`, `screen` или `pm2`:

```bash
# Через screen
screen -S step10bot
export BOT_TOKEN="твой_токен"
python bot.py
# Ctrl+A, D — отсоединиться (бот продолжит работать)
```
